# Capstone_VREdu
VR-based educational Capstone Project

VREdu is a Unity-based educational capstone project that aims to teach searching and sorting algorithms through a first-person 3D virtual classroom environment.

## Project Overview

The system is designed as an interactive learning application in which students explore a classroom containing algorithm stations and practice algorithms step by step using keyboard input. The project includes adaptive feedback, flashcard-based reinforcement, session tracking, and LLM integration for contextual learner support.

## Main Features

- Interactive 3D algorithm stations
- Searching and sorting algorithm practice
- Adaptive hints and feedback
- Flashcard-based revision
- Session summary and learner reflection
- LLM integration for contextual along with overall question answering

- ## Technology Stack

- Unity 2022.3 LTS
- C#
- TextMesh Pro
- Input System
- OpenAI API integration
- Windows PC target platform
